# Changelog

### [Public v1.0.0] - 2026-03-27
* **Major Overhaul:** Rebuilt the entire framework into a dedicated interactive shell.
* **Tiered Config:** Implemented the Global > Program > Target hierarchy to stop setting flags manually.
* **State Management:** Added a checkpoint system so the pipeline can resume after a crash/Ctrl+C.
* **Obsidian Integration:** Automated the conversion of raw tool output into structured Markdown notes.
* **Refined Pipeline:** Orchestrated 24 separate stages for a more tactical workflow.
* **Please Note:** This was public release 1.0.0 but internal 5.0, if you find any "5.0" references, let me know and I'll get them fixed.
